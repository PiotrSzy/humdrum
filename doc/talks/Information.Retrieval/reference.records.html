Humdrum Reference Records

Over 80 reference codes are pre-defined in Humdrum. Pertain to:
  • authorship information

  • performance information

  • work identification information

  • imprint information

  • copyright information

  • analytic information

  • representation information
COM:Composer's name
COA:Attributed composer
COS:Suspected composer
COL:Composer's abbreviated, alias, or stage name
COC:Composer(s) corporate name
CDT:Composer's dates
CNT:Nationality of the composer
LYR:Lyricist
LIB:Librettist
LAR:Arranger
LOR:Orchestrator
TXO:Original language of vocal/choral text
TXL:Language of the encoded vocal/choral text
TRN:Translator of text
MPN:Performer's name
MPS:Suspected performer
MRD:Date of performance
MLC:Place of performance
MCN:Name of the conductor of the performance
MPD:Date of first performance
OTL:Title
XEN:Translated title (in English)
XFR:Translated title (in French)
XDE:Translated title (in German)
XNI:Translated title (in Japanese)
OTP:Popular Title
OTA:Alternative title
OPR:Title of larger (or parent) work from which the encoded piece is a part
OAC:Act number
OSC:Scene number
OMV:Movement number
OMD:Movement designation or movement name
OPS:Opus number
ONM:Number
OVM:Volume
ODE:Dedication
OCO:Commission
OCL:Collector
ONB:Free format note related to the title or identity of the encoded work
ODT:Date of composition
OCY:Country of composition
OPC:City, town or village of composition
PUB:Publication status
PPR:First publisher
PDT:Date first published
PPP:Place first published
PC#:Publisher's catalogue number
SCT:Scholarly catalogue abbreviation and number
SCA:Scholarly catalogue (unabbreviated) name
SMS:Manuscript source name
SML:Manuscript location
SMA:Acknowledgement of manuscript access
YEP:Publisher of electronic edition
YEC:Date and owner of electronic copyright
YER:Date electronic edition released
YEM:Copyright message
YEN:Country of copyright
YOR:Year of copyright
YOO:Original document owner
YOY:Original copyright year
EED:Electronic Editor
ENC:Encoder of the electronic document
EMD:Document modification description
EEV:Electronic edition version
EFL:File number
EST:Encoding status
VTS:Checksum validation number
AFR:Form designation
AGN:Genre designation
AST:Style, period, or type of work designation
ASW:Associated Work
AMT:Metric Classification
AIN:Instrumentation
RLN:ASCII language setting
RDF:User-defined signifiers
RDT:Date encoded
RNB:Representation note
RWG:Representation warning
