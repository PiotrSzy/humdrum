SIGIR'99: Music Information Retrieval


How Themefinder Works


  • Web interface for the "thema" command:

    thema -T 3/8 -d 34554321 -C UDUDUD

    -T 3/8 option defines meter signature
    -d option defines scale degree search
    -C option defines gross up/down contour search

  • Can specify particular database for searching.

  • All files are flat ASCII.

  • Easy for users can add their own databases.

  • Searching is done by grep
