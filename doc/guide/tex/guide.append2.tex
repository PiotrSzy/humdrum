\section*{Instrumentation Codes}




\subsection*{Introduction}

\par 
Humdrum provides standard voice and instrument names in order
to facilitate various tasks.
Voice and instrument names are used both in
tandem interpretations, and in \texttt{AIN} reference
records identifying the instrumentation for a work (see
\textbf{Appendix I}).

\par 
Instrument tandem interpretations are used to identify the instrumentation
pertaining to a specified spine.
The word "instrument" is used in a broad sense and embraces
vocal qualities and vocal types as well as mechanical sound makers.



\par 
Three distinctions are currently made in Humdrum:
\textit{instrument name,}
\textit{instrument class}
and
\textit{instrument group.}
The following table identifies six pre-defined instrument classes:

\texttt{*ICvox}voice
\texttt{*ICstr}string instrument
\texttt{*ICww}woodwind instrument
\texttt{*ICbras}brass instrument
\texttt{*ICklav}keyboard instrument
\texttt{*ICidio}percussion instrument (idiophone)


\par 
The following table identifies five pre-defined instrument groups:

\texttt{*IGacmp}accompaniment instrument
\texttt{*IGsolo}solo instrument
\texttt{*IGcont}basso-continuo instrument
\texttt{*IGripn}ripieno instrument
\texttt{*IGconc}concertino instrument


\par 
The following set of tables list currently defined
instrument names.
When the name is used to form an instrument tandem
interpretation the instrument keyword is preceded by an asterisk
and the upper-case letter `\texttt{I}'.
For example, the instrument tandem interpretation for the guitar
is \texttt{*Iguitr}.
If the guitar appears as an instrument in a \texttt{AIN} Reference
Record only the designation `\texttt{guitr}' is used.
