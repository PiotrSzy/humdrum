HUMDRUM PACKAGE INSTALLATION

This document describes how to install the Humdrum Toolkit packages.

Introduction

This guide describes how to manually install the Humdrum Toolkit on your MacOS X or Red Hat Linux (or compatible) system.
 
  • If you are using MacOS X, you can download the Humdrum Toolkit package for MacOS X. 

    Once you have downloaded the package, you will need to unstuff it using the Expander program that came with MacOS X. (Your browser may automatically unstuff the archive after downloading it.)

    Double-click the package -- it should look like an open cardboard box -- and the installer should prompt you to complete the installation.

    Please read the documentation accompanying the installer for further information about completing the installation.

    •  

  • If you are using Red Hat Linux (or any other Linux system using RPM), you can download the Humdrum Toolkit RPM (currently under development).


Once you have finished the installation, you will probably want to visit the Quick-Start Tutorial.
